Boot sequence

From cold start to coding

Battery & Service

Request Doze exemption. Start foreground service with persistent notification.

Bootstrap Extraction

Extract Termux bootstrap zip into app's private storage. Full Linux userland, unpacked.

Runtime Installation

Install Node.js 24, Python, clang/cmake/make/lld for native module builds.

Agent Framework

Install Claw Code agent + Rust core. Build native FFI modules from source.

Proxy & Auth

Start CONNECT proxy for DNS/TLS. OAuth login via browser. One-time setup.

Launch

Gateway, Control UI server, web server — then WebView loads and you're coding.
